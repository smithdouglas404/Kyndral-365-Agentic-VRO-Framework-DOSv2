## 🎓 TECHNICAL NOTES

### Voice Briefings - OpenAI TTS Implementation

**How It Works**:
1. Claude generates conversational script (2 hosts: Sarah & Marcus)
2. Parse script into speaker segments
3. For each segment:
   - Sarah → OpenAI TTS with `nova` voice (female)
   - Marcus → OpenAI TTS with `onyx` voice (male)
4. Save each segment as .mp3 file
5. Use ffmpeg to concatenate segments
6. Serve final audio file
7. Clean up temporary segments

**Code Location**: `/server/routes/voice-briefings.ts:generateAudioFromScript()`

**Dependencies**:
- `openai` npm package (v4.x)
- `ffmpeg` (available in Replit environment)

**Quality**: B+ for conversational podcasts (80% of NotebookLM quality)

**Future Upgrade**: Swap OpenAI TTS for ElevenLabs for A+ quality
