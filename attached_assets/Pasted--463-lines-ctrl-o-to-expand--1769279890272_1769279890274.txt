  Example: SendGrid Setup                                                                                                             
                                                                                                                                      
  Step 1: Get API Key                                                                                                                 
  1. Go to https://app.sendgrid.com/                                                                                                  
  2. Sign up (free tier: 100 emails/day)                                                                                              
  3. Settings → API Keys → Create API Key                                                                                             
  4. Copy key (starts with "SG.")                                                                                                     
                                                                                                                                      
  Step 2: Add to Marketplace                                                                                                          
  Admin → MCP Marketplace → SendGrid                                                                                                  
  API Key: SG.xxxxxxxxxxxxx                                                                                                           
  From Email: noreply@yourcompany.com                                                                                                 
  [Test Connection] → ✓                                                                                                               
  [Save & Activate]                                                                                                                   
                                                                                                                                      
  Step 3: Assign to Agents                                                                                                            
  Admin → Agent Setup Wizard → Select agents                                                                                          
  Step 2: Check "SendGrid" for each agent                                                                                             
  [Save]                                                                                                                              
                                                                                                                                      
  Done! Agents can now send emails.
