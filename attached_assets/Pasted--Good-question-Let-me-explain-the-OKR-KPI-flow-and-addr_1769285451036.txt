  4. Complete Data Flow Example                                                                                                       
                                                                                                                                      
  1. SETUP (Wizard):                                                                                                                  
     ├─ Enable FinOps Agent                                                                                                           
     ├─ Load default OKRs: "Maintain profitability", "Budget control"                                                                 
     └─ Load default KPIs: CPI, SPI, Burn Rate                                                                                        
                                                                                                                                      
  2. DATA COLLECTION (Runtime):                                                                                                       
     ├─ Project Data (from Jira): Tasks, progress, team hours                                                                         
     ├─ Financial Data (from QuickBooks): Invoices, expenses, budgets                                                                 
     └─ FinOps calculates: CPI = Budget / Actual                                                                                      
                                                                                                                                      
  3. THRESHOLD MONITORING (Every 5 minutes):                                                                                          
     ├─ Check CPI: Current = 0.82                                                                                                     
     ├─ Compare to threshold: 0.82 < 0.85 (Critical!)                                                                                 
     └─ Trigger Camunda rule: "budget-overrun-critical"                                                                               
                                                                                                                                      
  4. CAMUNDA RULE EXECUTION:                                                                                                          
     ├─ Evaluate DMN table: "agent-collaboration.dmn"                                                                                 
     ├─ Decision: shouldCollaborate = true                                                                                            
     ├─ Target Agents: Risk, PMO, Governance                                                                                          
     └─ Actions: notify, email, escalate, attach_document                                                                             
                                                                                                                                      
  5. COLLABORATION:                                                                                                                   
     ├─ FinOps: Calculates recovery options                                                                                           
     ├─ Risk: Assesses financial risk impact                                                                                          
     ├─ Governance: Checks policy compliance                                                                                          
     └─ PMO: Prioritizes corrective actions                                                                                           
                                                                                                                                      
  6. NOTIFICATION:                                                                                                                    
     ├─ In-app alerts: All agents, executives                                                                                         
     ├─ Email: CFO, Project Leads                                                                                                     
     ├─ Dashboard: Red indicator on Executive Workspace                                                                               
     └─ Attached: "Budget Recovery Plan" from Knowledge Base
