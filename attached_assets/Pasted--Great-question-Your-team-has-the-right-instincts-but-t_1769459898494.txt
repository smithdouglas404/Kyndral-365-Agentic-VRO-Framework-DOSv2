  Tenant Model:                                                                                                                                                                                                                                                                                                                                            
                                                                                                                                                                                                                                                                                                                                                           
  System (Kyndryl Nexus)                                                                                                                                                                                                                                                                                                                                   
    └─ Tenant (Organization = One customer company)                                                                                                                                                                                                                                                                                                        
        └─ Company Profile (ACME Demo OR Real Company from 10-K)                                                                                                                                                                                                                                                                                           
            └─ Users (Multiple people, each with roles)                                                                                                                                                                                                                                                                                                    
                └─ Permissions (Admin, PMO, FinOps, Viewer, etc.)                                                                                                                                                                                                                                                                                          
                                                                                                                                                                                                                                                                                                                                                           
  User Roles:                                                                                                                                                                                                                                                                                                                                              
                                                                                                                                                                                                                                                                                                                                                           
  - System Admin (Kyndryl staff): Manage all tenants, view usage, provision                                                                                                                                                                                                                                                                                
  - Tenant Admin (Customer): Run setup wizard, invite users, configure company                                                                                                                                                                                                                                                                             
  - PMO User: Access PMO dashboards                                                                                                                                                                                                                                                                                                                        
  - FinOps User: Access FinOps dashboards                                                                                                                                                                                                                                                                                                                  
  - Viewer: Read-only access                                                                                                                                                                                                                                                                                                                               
                                                                                                                                                                                                                                                                                                                                                           
  Registration Flow (Self-Service):                                                                                                                                                                                                                                                                                                                        
                                                                                                                                                                                                                                                                                                                                                           
  1. User clicks "Start Free Trial"                                                                                                                                                                                                                                                                                                                        
  2. Enters email + password → Email verification                                                                                                                                                                                                                                                                                                          
  3. Creates organization (tenant) name: "Acme Corp"                                                                                                                                                                                                                                                                                                       
  4. Choose: "Try Demo" (ACME) OR "Setup My Company" (10-K extraction)                                                                                                                                                                                                                                                                                     
  5. Lands in dashboard                                                                                                                                                                                                                                                                                                                                    
  6. Can invite teammates via email                                                                                                                                                                                                                                                                                                                        
                                                                                                                                                                                                                                                                                                                                                           
  Admin Provisioning Flow (Sales-Led):                                                                                                                                                                                                                                                                                                                     
                                                                                                                                                                                                                                                                                                                                                           
  1. Prospect requests demo → Sales qualifies                                                                                                                                                                                                                                                                                                              
  2. You (System Admin) create tenant: "Acme Corp"                                                                                                                                                                                                                                                                                                         
  3. Add customer's email as Tenant Admin                                                                                                                                                                                                                                                                                                                  
  4. They get email invitation → Set password                                                                                                                                                                                                                                                                                                              
  5. First login → Setup Wizard (Demo or Real)                                                                                                                                                                                                                                                                                                             
  6. They can invite their team
